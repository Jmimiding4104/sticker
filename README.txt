1.請下載精臣雲打印電腦版(https://www.niimbot.com/cnweb/pc.html)，後安裝完成並打開程式。

2.開啟診篩列印貼紙

3.開啟讀取健保卡API


診篩列印貼紙

1.若有健保卡可插入後點擊「獲取健保卡資料」即可取得姓名、身分證字號、生日。

2.若健保卡資訊有無法讀取的，可修改程式上文字。

3.教育點擊即可(單選)。

4.電話請詢問個案。

5.地址目前預設桃園市龍潭區，此預設字可自行調整，下方有里別之快捷鍵，只需要點擊可自動帶入里別之罐頭訊息。

6.選擇施作項目，右方會自動累加數量做參考。

7.確認無問題後點擊提交，下方即會呈現資料內容可預覽，此時文字已複製於電腦中，直接至打印機軟體貼上即可。


精臣打印機

1.需先連結打印機本體，請使用 typeC 線做有線連結。

2.若上方顯示未連結請點擊該區塊，做連結確認。

3.連結完成後點擊左方新建標籤，可自動辨識標籤紙(T50*20-320白色)。

4.點擊左邊文本，將中間編輯區塊拉至適當的寬度(只需調整寬度，高度自適應)。

5.文字大小最佳為9號，可自行調整大小。

6.將文字內容複製貼上完成之文字資料。

7.點擊列印。

8.選擇張數。

9.完成，請接下來下一位。
